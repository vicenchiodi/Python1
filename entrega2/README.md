# Entrega 2

Proyecto desarrollado en Jupyter Notebook.

## Requisitos previos

Tener instalado *Python 3* en tu sistema. Podés verificarlo ejecutando:

bash
python --version


## Instalación de Jupyter Notebook

bash
pip install notebook


> Si tenés múltiples versiones de Python instaladas, usá pip3 en lugar de pip.

## Ejecución

1. Cloná o descargá el repositorio en tu máquina local.

2. Abrí una terminal y navegá hasta la carpeta del proyecto:

bash
cd Python1/entrega2/notebook


3. Iniciá Jupyter Notebook:

bash
jupyter notebook


4. Se abrirá automáticamente el navegador con la interfaz de Jupyter. Desde ahí, hacé clic en *notebook.ipynb* para abrirlo.

5. Ejecutá las celdas en orden con *Shift + Enter* o usá el menú *Kernel → Restart & Run All* para ejecutar todo de una vez.
